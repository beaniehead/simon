RULES

  1. The strict button toggles strict mode. In strict mode, one mistake starts the game over!
  2. Press Start. Simon will give the first signal. Repeat the signal by selecting the same color.
  3. Each turn Simon will add one more color to the sequence.
  4. Repeat the sequence to progress.Simon will repeat a sequence if you make a mistake.
  5. Complete all the rounds to win.
The game gets faster the further you progress.
